# TravelSystem Module

Contract-based traveling merchants.

## Features
- Create merchant contracts
- Merchants travel between towns
- State machine for lifecycle
- Automatic return & settlement

## Commands
- `/contract create` - Create contract
- `/contract list` - List active contracts
- `/contract cancel` - Cancel contract

## Dependencies
- Core (required)
- Merchants Module (required)
- Storage Module (required)
