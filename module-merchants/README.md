# Merchants Module

NPC-based trading system.

## Features
- Spawn merchants on plots
- Define trade offers
- Inventory management
- Economy integration

## Commands
- `/merchant create` - Spawn merchant
- `/merchant offer add` - Add trade offer
- `/merchant remove` - Remove merchant

## Dependencies
- Core (required)
- Storage Module (required)
- NPCProvider (required)
- EconomyProvider (required)
