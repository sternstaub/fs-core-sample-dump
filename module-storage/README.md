# Storage Module

Chest-based storage system for plots/towns.

## Features
- Register chests on plots
- Material-specific tracking
- Sneak + right-click assignment
- Plot-based OR player-based (fallback)

## Commands
- `/storage register` - Register chest
- `/storage list` - List registered chests
- `/storage info` - View storage stats

## Dependencies
- Core (required)
- PlotProvider (optional)
