# AdminShops Module

Template-based admin shops with dynamic pricing.

## Features
- YAML-based templates
- Item pools
- Dynamic pricing (based on player shops)
- Infinite stock

## Commands
- `/adminshop create <template>` - Create admin shop
- `/adminshop reload` - Reload templates

## Dependencies
- Core (required)
- Merchants Module (required)
